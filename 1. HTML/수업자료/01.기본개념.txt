Client : 클릭한 페이지를 요청하는 pc

Server : 클릭된 페이지를 제공하는 pc
=> 반드시 대규모 서버만 있는것은 아님

Client 의 요청(request) 를 받아
ex) 네이버 페이지 띄워줘 (www.nave.com) 을 입력

Server 에서 응답 (response) 한다.
=> HTML 문서로 답을 보낸다.

==> 이를 예쁘게 꾸며서 보여주는게 웹 브라우저
ex) 구글, 파이어폭스 ..
-------------------------------------------------------------
HTML 뼈대
ex) 기반을 닦아 건물을 지음

CSS 디자인
ex) 내외부 인테리어를 해서 예쁘게 꾸며줌

JS 기능
ex) 불을 키고 끄는 등의 기능적인 것들을 추가해줌
-------------------------------------------------------------
