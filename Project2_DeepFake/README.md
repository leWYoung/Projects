## 👀 서비스 소개
* 서비스명:  딥러닝 모델을 활용한 딥페이크 분석 서비스
* 서비스설명: 딥러닝 모델을 활용하여 실시간으로 딥페이크 여부를 판단하고
 <br>사용자에게 분석 결과를 제공하는 서비스입니다. 
<br>

## 📅 프로젝트 기간
2024.11.04 ~ 2024.12.26 (8주)

* 음성 분석 기능
* 분석 결과 PDF 저장 기능
* 유관기관 연결 URL
